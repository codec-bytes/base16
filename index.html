    @codec-bytes/base16

    Base16 encoder/decoder for JavaScript. See docs. Parent is @codec-bytes/about.

    import {encode, decode} from '@codec-bytes/base16';
encode('DEADBEEF'); // [ 222, 173, 190, 239 ]
decode([ 222, 173, 190, 239 ]); // 'DEADBEEF'
